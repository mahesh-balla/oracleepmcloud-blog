## Oracle EPM Cloud 25.04 Release Overview

April 2025 delivers significant user experience improvements, powerful new calculation capabilities, and major platform migration milestones. The 25.04 release focuses on making EPM Cloud more intuitive, more capable, and better positioned for enterprise-scale deployments on OCI infrastructure.

## User Experience Enhancement

### Refreshed Navigation Experience

The 25.04 release introduces a **redesigned navigation panel on the left side** of the interface, fundamentally improving how users move between dashboards, forms, and tasks. The new navigation panel provides:

- **Simplified Dashboard Switching**: Quick access to frequently used dashboards
- **Form Organization**: Streamlined form discovery and navigation
- **Task Management**: Easy access to task workflows and automation
- **Improved Visual Organization**: Cleaner, more intuitive layout that reduces cognitive load

This refreshed navigation experience makes it faster and easier for users to accomplish their work within EPM Cloud.

## Calculation Manager Enhancements

### Dynamic Substitution Variables

Calculation Manager receives a powerful new capability in 25.04: **dynamic substitution variables in calculation rules**. This enhancement enables organizations to:

- **Create More Versatile Rules**: Build rules that adapt to different scenarios without requiring multiple versions
- **Improve Rule Reusability**: Write calculations once and deploy them across multiple contexts
- **Reduce Maintenance Burden**: Simplify rule management by eliminating duplicate rule definitions
- **Enhance Flexibility**: Support complex calculation scenarios that previously required manual workarounds

Dynamic substitution variables represent a significant step forward in calculation flexibility and reduce the overhead of managing large rule libraries.

## Security Enhancements

The 25.04 release strengthens security across multiple modules:

- **HTTPS Mandated**: All connections in Enterprise Data Management (EDM), Data Integration (DI), Task Manager, and Narrative Reporting now require HTTPS
- **TLS 1.3 Support**: Enhanced encryption protocol support for stronger data protection
- **Encryption in Transit**: All sensitive data is encrypted during transmission

These security enhancements ensure that sensitive financial and planning data remains protected throughout its lifecycle in the cloud.

## Data Integration Improvements

### Enhanced Integration Capabilities

Data Integration receives several important improvements in 25.04:

- **Copy Existing Integrations**: Duplicate Data Integration pipelines to streamline setup of similar integrations
- **Custom Labels for Missing Data**: Define custom labels and handling for missing data scenarios
- **Disable Implied Shares for BSO**: Greater control over Block Storage Cube behavior and data sharing rules

### OCI Migration Achievement

A major milestone in 25.04 is the **automatic cloning of Classic environments to OCI**. Organizations running Classic environments will have their applications automatically migrated and cloned as part of the 25.04 update, reducing manual migration effort and accelerating the transition to modern OCI infrastructure.

## GenAI Insights in Intelligent Performance Management

### Insight Summarization

The **Intelligent Performance Management module gains GenAI-powered insight summarization**. This new capability:

- **Analyzes Performance Data**: Uses artificial intelligence to identify key trends and anomalies
- **Generates Executive Summaries**: Creates concise summaries of performance insights
- **Accelerates Decision-Making**: Enables faster identification of issues and opportunities
- **Improves Visibility**: Provides clearer understanding of performance drivers

GenAI-powered insights help finance teams focus on strategic analysis rather than manual data interpretation.

## Smart View Integration with Google Workspace

### Query Reports in Google Sheets

The Smart View for Google Workspace integration is further enhanced in 25.04, enabling users to:

- **Run Query Reports**: Execute ad-hoc queries directly from Google Sheets
- **Access Cloud EPM Data**: Seamlessly connect to EPM Cloud data sources
- **Maintain Familiar Workflows**: Work within Google's productivity environment
- **Support Collaborative Analysis**: Leverage Google Sheets' collaboration features for team analysis

This integration continues to expand the platform's reach beyond traditional EPM interfaces.

## Planning for the Transition

Organizations should prepare for the 25.04 update by:

- **Training Teams**: Familiarize users with the new left-side navigation panel
- **Testing Calculations**: Validate existing rules before the update
- **Planning OCI Integration**: Understand the automatic migration process and any required post-migration steps
- **Evaluating GenAI Insights**: Consider how to incorporate GenAI analysis into decision-making workflows
